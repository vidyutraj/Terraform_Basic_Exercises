# Terraform Basic Exercises

## Project Overview
This repository contains basic exercises to help individuals learn and practice working with Terraform, a popular Infrastructure as Code (IaC) tool.

## Project Structure
The project is structured as follows:

```
Terraform_Basic_Exercises/
├── Exercise_1/
│   ├── main.tf         # Main configuration file for Terraform
│   └── variables.tf    # Variable definitions
├── Exercise_2/
│   ├── main.tf         # Main configuration file for Terraform
│   └── outputs.tf      # Outputs from the configuration
└── Exercise_3/
    ├── main.tf         # Main configuration file for Terraform
    └── README.md       # Documentation for this exercise
```

### Exercise Directories
- **Exercise_1/**: Basic setup related to Terraform, introducing core concepts.
- **Exercise_2/**: Intermediate exercises that involve outputs and state management.
- **Exercise_3/**: Advanced configurations and usage examples.

## Purpose
The primary purpose of this repository is to facilitate learning and to provide practical examples of how to use Terraform in various scenarios. Each exercise includes instructions and examples to demonstrate different aspects of Terraform.

## Getting Started
To get started with the exercises:
1. Clone the repository: `git clone https://github.com/vidyutraj/Terraform_Basic_Exercises.git`
2. Navigate to the desired exercise folder.
3. Initialize Terraform: `terraform init`
4. Apply the configuration: `terraform apply`
