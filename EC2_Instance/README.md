# README for EC2 Instance

## Overview
This README provides information regarding the EC2 Instance setup using Terraform.

## Prerequisites
Before you begin, ensure you have the following:
- AWS account
- Terraform installed

## Setting Up EC2 Instances
To set up EC2 instances, follow these steps:
1. **Clone the repository**: `git clone https://github.com/vidyutraj/Terraform_Basic_Exercises.git`
2. **Navigate to the directory**: `cd Terraform_Basic_Exercises/EC2_Instance`
3. **Initialize Terraform**: Run `terraform init` to initialize the working directory.
4. **Plan the deployment**: Execute `terraform plan` to see the resources that will be created.
5. **Apply the changes**: Run `terraform apply` to create the specified EC2 instances.

## Resource Management
Use the following commands for managing your Terraform resources:
- `terraform status`: Check the current state of the resources.
- `terraform destroy`: Delete all the resources specified in the configuration.

## Conclusion
This concludes the setup process for EC2 instances. Review AWS console for your resources.
