# Terraform Basic Exercises

## Overview
This repository contains basic exercises using Terraform to set up and manage AWS EC2 instances.

## Prerequisites
- [ ] AWS Account
- [ ] Terraform installed
- [ ] AWS CLI configured

## File Descriptions
| File Name             | Description                                          |
|----------------------|------------------------------------------------------|
| `main.tf`           | Main configuration file for Terraform defining AWS resources.
| `variables.tf`      | Defines the variables used in the Terraform scripts.
| `output.tf`         | Outputs the metadata for the created AWS resources.
| `README.md`         | Documentation for the repository.
